# Flip-Flop and Counter Design using Verilog HDL

## Project Overview

This project implements a D Flip-Flop and a 4-Bit Synchronous Counter using Verilog HDL. The designs were simulated and verified using EDA Playground and EPWave.

## Objectives

* Design a D Flip-Flop using Verilog HDL.
* Design a 4-Bit Counter using Verilog HDL.
* Verify functionality through simulation and waveform analysis.
* Understand sequential circuit design in VLSI systems.

## Tools Used

* Verilog HDL
* EDA Playground
* EPWave
* GitHub

## Results

### D Flip-Flop

The D Flip-Flop successfully transfers the input D to output Q on the rising edge of the clock.

### 4-Bit Counter

The counter increments sequentially from 0000 to 1111 on each rising clock edge and resets correctly when the reset signal is asserted.

## Conclusion

The project demonstrates the design and simulation of basic sequential circuits used in VLSI systems. The functionality of both designs was verified through waveform analysis.
